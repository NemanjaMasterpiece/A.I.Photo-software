This project is an AI-based photo classification software developed in Python using Jupyter Notebook, TensorFlow, and Keras.
The software uses a Convolutional Neural Network (CNN) to classify images of cats and dogs.

Features:

    - Dataset preprocessing and augmentation

    - CNN model architecture design

    - Model training and evaluation

    - Image prediction on new input data

Technologies:

    - Python

    - TensorFlow / Keras

    - NumPy, Matplotlib

    - Jupyter Notebook

How to Run:

1. Clone this repository

2. Install required packages:

	pip install -r requirements.txt

3. Run in Jupyter Notebook:

	jupyter notebook

4. Open A.I. Photo software.ipynb and execute all cells.
